🚑 Pratham-Chikitse – First Aid Emergency Offline Guide

📌 Project Overview

Pratham-Chikitse is an Android-based healthcare emergency assistance application developed using Android App Development with GenAI concepts. The application provides medically accurate first-aid guidance for common emergencies such as snake bites, burns, choking, fractures, heart attacks, and other critical situations.

The app is designed to work completely offline, making it highly useful in rural areas, remote locations, and during network failures. It provides step-by-step emergency instructions in both Kannada and English along with audio guidance using Text-To-Speech technology.

---

🎯 Problem Statement

In emergency situations, the first few minutes are extremely critical. Many people panic or follow incorrect home remedies due to the lack of medically accurate guidance in their local language.

Pratham-Chikitse aims to solve this problem by providing:

* Accurate first-aid instructions
* Offline emergency support
* Kannada and English language support
* Audio-based emergency guidance
* Easy-to-use emergency interface
